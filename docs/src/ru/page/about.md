# О демо

Это демонстрационный сайт темы
[vitepress-theme-neptu-blog](https://github.com/bozonx/vitepress-theme-neptu-blog).
Каждый пост и страница показывают реальную возможность темы, а рядом — исходный
frontmatter или конфиг.

Наиболее полная документация находится в **английской локали** — переключите
язык в верхней панели. Здесь, в русской локали, показан рабочий пример
мультиязычного блога: своя главная, свои ленты, sitemap и все типы списков
(свежие, популярные, архив, авторы, теги).

## Запуск локально

```bash
pnpm install
pnpm example:dev
```
