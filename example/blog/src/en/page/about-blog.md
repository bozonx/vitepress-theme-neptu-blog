# About This Blog

This is the example blog that ships with
[vitepress-theme-neptu-blog](https://github.com/).
It serves three purposes simultaneously:

1. **Demo** — shows what the theme looks like with real content.
2. **Scaffold** — copy this directory to start your own blog.
3. **E2E fixture** — the posts cover the full range of frontmatter options
   so they can be used as test fixtures.

## Getting started

Copy the `example/blog` directory, run `pnpm install`, then `pnpm dev`.
