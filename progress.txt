TypeScript migration progress
=============================

Strategy: Strict + as-source distribution (matches VitePress 2 default theme).
- tsconfig.json: strict, allowJs (gradual), noEmit, moduleResolution: bundler.
- vue-tsc --noEmit is the type checker (run `pnpm typecheck`).
- exports map in package.json keeps pointing to source files; extensions are
  updated from .js -> .ts as each module is migrated.

Rules during migration:
- No behavioural changes; only typing.
- Public API of each module preserved (named exports, signatures).
- Use `import type` for type-only imports.
- For SFC: defineProps<...>(), withDefaults(), defineEmits<...>().
- For Node-only modules (transformers, helpers used in config.ts), no DOM types.
