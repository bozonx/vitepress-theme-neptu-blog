TypeScript migration progress
=============================

Strategy: Strict + as-source distribution.
- tsconfig.json: strict, allowJs, noEmit, moduleResolution: bundler.
- vue-tsc --noEmit is the type checker (`pnpm typecheck`).
- exports map points to source files; extensions update from .js -> .ts as
  each module is migrated.

Rules during migration:
- No behaviour changes; only typing.
- Public API of each module preserved (named exports, signatures).
- Use `import type` for type-only imports.
- For SFC: defineProps<...>(), withDefaults(), defineEmits<...>().
- For Node-only modules, no DOM types.
